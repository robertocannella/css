# CSS Flex Properites & Values

## flex value
* adding ```flex``` value for ```display``` property on a containing div creates a parent container
```
.flex-container {
    background: white;
    padding: 10px;
    border: 5px solid black;    
    display: flex;
}
```
## flex-inline value
* The ```flex-inline``` value does not allow resizing of specified widths
```
.flex-container {
    background: white;
    padding: 10px;
    border: 5px solid black;    
    display: flex-inline;
}
